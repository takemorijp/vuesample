
  # サイドバーとヘッダとボディのサンプル画面

  サイドバーとヘッダとボディのサンプル画面作成（reactGamen5）

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
## Notes for Vue-converted version

This folder has been converted from a React starter to a minimal Vue 3 scaffold for experimentation.

- Start dev server:

```cmd
cd reactGamen5
npm install
npm run dev
```

- The app entry is `src/main.ts` and the primary component is `src/App.vue`.
- Global styles are loaded from `src/styles/index.css` (which imports `tailwind.css`, `fonts.css`, and `theme.css`).

If you encounter PowerShell execution policy errors when running npm scripts, run the commands in CMD instead of PowerShell.
